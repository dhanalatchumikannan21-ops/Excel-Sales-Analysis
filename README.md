Retail Sales Performance Analysis Dashboard
Project Overview

This project was developed using Microsoft Excel to analyze retail sales performance across different regions, categories, and product sub-categories.

The objective was to transform raw sales data into meaningful business insights through data cleaning, Pivot Tables, Pivot Charts, KPIs, and interactive dashboard elements.

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Dashboard Design
Dataset Information

The dataset contains approximately 10,000 sales records and includes:

Region
Category
Sub-Category
Sales
Profit
Quantity
Customer Information
Project Workflow
1. Data Preparation
Imported retail sales dataset
Converted raw data into Excel Table format
Verified data structure and prepared it for analysis
2. Pivot Table Analysis

Created Pivot Tables to analyze:

Sales by Region
Profit by Category
Sales by Sub-Category
Quantity Sold by Category
3. Dashboard Development

Created an interactive dashboard containing:

Total Sales KPI
Total Profit KPI
Total Quantity KPI
Pivot Charts
Slicers for dynamic filtering
Key Insights
West region generated the highest sales.
Technology was the most profitable category.
Phones were the top-selling sub-category.
Office Supplies contributed the highest quantity sold.
